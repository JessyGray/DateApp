# 💖 Date App

**Catch the Button Date App**

An interactive romantic invitation web application featuring animations, a playful runaway button, date selection, and Telegram notifications.

---

# 🇬🇧 English

## ✨ About

Date App is a small interactive invitation website.

The project combines romantic design, glassmorphism effects, animations, and a playful "catch the button" experience.

Desktop users must catch the "Not now" button, while mobile users receive a simpler interaction.

---

## 🚀 Features

* Glassmorphism user interface
* Animated background
* Responsive design
* Interactive runaway button
* Date selection
* Time selection
* Meeting format selection
* Final confirmation screen
* Telegram notifications
* Desktop and mobile optimized

---

## 💻 Desktop Experience

* "Not now" button runs away.
* Button moves across the entire screen.
* Mouse interaction supported.
* Random escape behavior.

---

## 📱 Mobile Experience

* Buttons appear vertically.
* "Not now" button remains visible.
* Small click animation only.
* No accidental rejection.
* Optimized for phones and tablets.

---

## 🔔 Telegram Notifications

The application can send Telegram messages when:

* User presses "Yes"
* User presses "No"

### Change Telegram settings

Open the JavaScript section:

```javascript
const token = "YOUR_BOT_TOKEN";
const chatId = "YOUR_CHAT_ID";
```

Replace them with your own values.

You can create a Telegram bot using:

* @BotFather
* @userinfobot

# 🇷🇺 Русская версия

## ✨ О проекте

Date App — это небольшое интерактивное приглашение на встречу.

Проект сочетает романтический дизайн, стеклянный интерфейс, анимации и игру с убегающей кнопкой.

---

## 🚀 Возможности

* Glassmorphism интерфейс
* Анимированный фон
* Адаптивный дизайн
* Убегающая кнопка
* Выбор даты
* Выбор времени
* Выбор формата встречи
* Финальная страница
* Уведомления в Telegram
* Поддержка ПК и мобильных устройств

---

## 💻 Версия для компьютеров

* Кнопка «Не сейчас» убегает.
* Кнопка перемещается по всему экрану.
* Поддерживается управление мышью.

---

## 📱 Мобильная версия

* Кнопки располагаются вертикально.
* Кнопка «Не сейчас» остаётся видимой.
* Только небольшая анимация нажатия.
* Случайный отказ исключён.

---

## 🔔 Уведомления Telegram

Приложение может отправлять сообщения в Telegram:

* При нажатии «Да»
* При нажатии «Не сейчас»

### Как изменить настройки Telegram

Откройте JavaScript-код:

```javascript
const token = "YOUR_BOT_TOKEN";
const chatId = "YOUR_CHAT_ID";
```

Укажите собственный токен и ID.

Получить их можно через:

* @BotFather
* @userinfobot
